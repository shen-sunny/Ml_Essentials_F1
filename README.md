# Ml_Essentials_F1

# F1 Podium Prediction — Interpretable ML

Predicts whether an F1 driver will finish on the podium (top 3) using Ergast historical data.

## Files
- `f1_podium_prediction.ipynb` — main notebook
- `data/` — Ergast CSV files (results, qualifying, races, drivers, constructors)
- `presentation/` — project slides

## How to run
1. `pip install pandas numpy scikit-learn matplotlib seaborn`
2. Open and run `f1_podium_prediction.ipynb`
